# Dashboard Template

Full template for `Dashboard.md` with all required sections.

## Template

```markdown
# Dashboard

> Last updated: YYYY-MM-DDTHH:MM:SSZ

---

## Pending Actions

| Status | Item | Summary | Priority | Date Added |
|--------|------|---------|----------|------------|
| [ ] | [[Needs_Action/FILENAME]] | One-line summary | #priority | @YYYY-MM-DD |

> Total pending: N items

---

## Recently Completed

| Item | Summary | Completed |
|------|---------|-----------|
| [[Done/FILENAME]] | One-line summary | @YYYY-MM-DD |

> Showing last 10 completed items

---

## Stats

| Folder | Count |
|--------|-------|
| Inbox | N |
| Needs_Action | N |
| Done | N |
| Plans | N |
| Pending_Approval | N |
| Approved | N |
| Logs | N |

> Counts reflect actual folder contents at time of update

---

## Alerts

| Alert | Item | Age | Priority |
|-------|------|-----|----------|
| OVERDUE | [[Needs_Action/FILENAME]] | Xh | #high |

> Items listed here are #high priority and older than 24 hours

---

*Auto-generated by Digital FTE. Do not edit manually.*
```

## Section Rules

### Pending Actions
- Source: Read all files in `/Needs_Action/`
- Sort: `#high` first, then `#medium`, then `#low`; within same priority, oldest first
- Format: Table row per item with checkbox, wiki-link, summary, priority tag, date

### Recently Completed
- Source: Read all files in `/Done/`, sort by completion date descending
- Limit: Show only the 10 most recent
- Format: Table row per item with wiki-link, summary, completion date

### Stats
- Source: Count files in each of the 7 folders
- Method: **Always count actual files**, never cache or estimate
- Format: Simple two-column table

### Alerts
- Source: Filter `/Needs_Action/` items where:
  - Priority is `#high` AND
  - Date added is more than 24 hours ago
- Sort: Oldest first (most overdue at top)
- Format: Table with alert type, wiki-link, age in hours, priority

## Timestamp Format

All timestamps must use ISO-8601: `YYYY-MM-DDTHH:MM:SSZ`

Examples:
- `2026-02-19T14:30:00Z`
- `2026-02-19T09:00:00Z`

## Empty State

When a section has no items, display:

```markdown
## Pending Actions

> No pending actions.

## Recently Completed

> No completed items yet.

## Alerts

> No active alerts.
```
