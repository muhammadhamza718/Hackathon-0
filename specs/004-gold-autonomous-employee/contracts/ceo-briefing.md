# Contract: CEO Briefing Engine

**Skill**: `ceo-briefing-engine`
**Module**: `agents/gold/briefing_engine.py`

## Interface

### `CEOBriefingEngine`

#### `__init__(odoo_client: OdooRPCClient, vault_root: Path, config: BriefingConfig)`
- **Precondition**: Odoo client initialized (may or may not be authenticated — handle gracefully).
- **Postcondition**: Engine ready to generate briefings.

#### `should_generate() -> bool`
- **Behavior**: Check if current time matches scheduled trigger (Sunday, 22:00 local).
- **Logic**: `weekday == config.briefing_day AND hour >= config.briefing_hour AND no briefing exists for current ISO week`.
- **Returns**: True if briefing should be generated now.

#### `generate_briefing() -> CEOBriefing`
- **Behavior**: Aggregate data from all sources and produce a briefing.
- **Data aggregation paths**:
  1. **Revenue MTD**: Odoo `account.move` (out_invoice, posted, current month) → sum `amount_total`. Compare with `revenue_goal` from `Company_Handbook.md` or env fallback.
  2. **Bottlenecks**: Scan `/Needs_Action/` for files where `mtime > 48 hours ago`. Parse priority from content. Rank by priority then age.
  3. **Subscriptions**: Read subscription registry from `Company_Handbook.md`. Check utilization (<30% = underused). Check market rate (>20% above = overpriced). Calculate potential savings.
- **Graceful degradation**: If Odoo is unreachable, revenue section shows `[DATA UNAVAILABLE]`. If Company_Handbook lacks subscription data, that section shows `[DATA UNAVAILABLE]`.
- **Returns**: `CEOBriefing` with all available data.
- **Side effects**: Logs `ceo_briefing` audit entry.

#### `write_briefing(briefing: CEOBriefing) -> Path`
- **Behavior**: Render briefing as Markdown and write to `/Needs_Action/`.
- **Filename**: `CEO-Briefing-YYYY-WNN.md` (ISO week).
- **Returns**: Path to the written file.
- **Side effects**: Updates Dashboard.md.

### `BriefingConfig`

```python
@dataclass(frozen=True)
class BriefingConfig:
    briefing_day: int = 6       # 0=Mon, 6=Sun
    briefing_hour: int = 22     # 24h format
    revenue_goal: float = 10000.0  # Fallback if not in Company_Handbook
    bottleneck_threshold_hours: float = 48.0
    utilization_threshold_pct: float = 30.0  # Below this = underused
    overpriced_threshold_pct: float = 20.0   # Above market rate by this = overpriced
```
